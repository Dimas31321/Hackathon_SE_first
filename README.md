Все письма будем распределить на 8 категорий:
1) Спам / Spam
2) Черновики / Draft
3) Сбои (обращение в техподдержку) / Failure
4) Для перенаправления (просмотреть) / Redirect
5) Заявки (Где есть ID заявки и не только) / Applications
6) Неверный формат (Картинка, например) / Wrong format
7) Требуют ответа / Requires an answer
8) Другие / Other

Категории хранятся в массиве класса, в переменой categories
