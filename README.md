Программа распределяет письма по следующим 8 категориям:
1) Спам
2) Черновики
3) Сбои
4) Для перенаправления
5) Заявки
6) Неверный формат
7) Требуют ответа
8) Другие

Распределяются письма из директории inbox

Ключевые слова для категорий хранятся в директории types, так-же можно добавлять отправителей в категории, для этого нужно пропустить пустую строку и прописать адреса отправителей.

Письма распределяются по категориям в директории out.

## Как запустить

Программа запускается через docker compose.

Реализованны две версии. Одна основывается только на ключевых словах и адресах отправителей. Запускается она через следующую команду.

`docker compose run lite_version`

Расширенная версия помимо основного распределения по категориям предоставляет возможность поиска подходящих по смыслу к запросу писем, для этого она использует эмбеддинги. Она запускается через команду.

`docker compose run extended_version`
